# 🌙 Grand Iftar Party 2026 - Countdown Dashboard

বন্ধুদের ইফতার পার্টির জন্য একটি প্রিমিয়াম এবং ইউনিক HTML রিমাইন্ডার ড্যাশবোর্ড। এতে রয়েছে লাইভ কাউন্টডাউন, নিওন গ্লাস-মরফিজম ডিজাইন এবং RGB অ্যানিমেশন।

## ✨ ফিচারসমূহ:
* **বিসমিল্লাহির রাহমানির রাহিম:** সবার উপরে বরকতময় শুরু।
* **ডুয়েল কাউন্টডাউন:** ইফতারের বাকি সময় এবং দুপুর ৩টার রিপোর্টিং টাইমের আলাদা ঘড়ি।
* **লাইভ সময়:** রিয়েল-টাইম সেকেন্ডসহ ঘড়ি।
* **RGB অ্যানিমেশন:** বক্সগুলোতে ডায়নামিক বর্ডার লাইটিং।
* **ব্রেকিং নিউজ:** নিচে বন্ধুদের নামসহ মজার স্ক্রলিং হেডলাইন।
* **স্মার্ট ওয়ার্নিং:** ১০ সেকেন্ড পর পর পপ-আপ মেসেজ।
